## New Features

### Buttons in Quick Responses

Text quick responses can now include interactive buttons: up to **3 reply buttons** (the customer taps to reply instantly) or **one website button** (opens a link). Build them right in the quick response editor with a live preview.

When you pick a quick response with buttons in the chat inbox, a confirmation window opens so you can review — and edit — the message before it's sent. Nothing goes out automatically.

### Buttons in the API

The [Send Free-Form Message](/api/messages/send-free-form-message) endpoint now accepts a `buttons` object, so you can send the same interactive buttons programmatically.
